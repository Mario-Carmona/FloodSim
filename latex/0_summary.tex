\subsection*{3. Simulation Dataset \& Geospatial Data Credits}
The baseline dataset used for the simulation environment is a dynamically generated derived work, compiled via an automated pipeline and licensed under a \textbf{Creative Commons Attribution 4.0 International License} (CC BY 4.0). 

\begin{figure}[ht]
    \centering
	\includegraphics[scale=1]{images/licences/CC_BY_4_0_license.png}
\end{figure}

\noindent This dataset aggregates and processes raw information from the following public data sources:
\begin{itemize}
    \item \textbf{Digital Elevation Model (MDT05):} Derived work from IGN / Sistema Cartográfico Nacional (España). Licensed under CC BY 4.0 (\textit{scne.es}).
    \item \textbf{Land Cover System (SIOSE AR):} Derived work from sioseAR 2020, provided by IGN / Sistema Cartográfico Nacional (España). Licensed under CC BY 4.0.
    \item \textbf{Rainfall Data (SAIH):} Sourced from the \textit{Sistema Automático de Información Hidrológica} - Confederación Hidrográfica del Júcar (CHJ). Used under Ley 37/2007 (CC BY 4.0 equivalent).
    \item \textbf{Digital Bathymetry (DTM):} Sourced from the EMODnet Bathymetry Consortium / European Union (\copyright~1995-2026). Licensed under CC BY 4.0.
\end{itemize}
