- Modelo Tensorflow:

Optimizar tensorflow. Al cargar el modelo, inicializar el estado de las capas estáticas y del dx y dt. 
En cada step se le pide al modelo que genere el nuevo estado, pero este nuevo estado no lo tiene que 
obtener el core, simplemente se le asigna el nuevo estado a la variable que tenía el estado anterior 
de esta forma evitamos pasar información innecesaria del modelo al core. El core solo lee los outputs 
del modelo cuando le toca generar el snapshot. De esta forma se reduce al máximo la comunicación entre 
modelo y core
